# Google Play Console 設定ガイド

Tsuzumi ストア掲載時に Console で入力する内容をまとめたドキュメント。

## 1. ストアの掲載情報

| 項目 | 内容 |
|------|------|
| アプリ名 | Tsuzumi |
| 短い説明 | `store-listing/short-description.txt` を貼り付け |
| 詳しい説明 | `store-listing/full-description.txt` を貼り付け |
| アプリアイコン | AABから自動取得 |

| スクリーンショット | `store-listing/` の PNG（推奨掲載順は下記） |

## 5. データセーフティ

`docs/google-play-data-safety.md` を参照して回答。

### 要点
- データ収集: あり（匿名UID、習慣データ、クラッシュログ）
- データ共有: なし（第三者への販売・共有なし）
- 暗号化: あり（HTTPS）
- 削除リクエスト: あり（アプリ内「アカウント削除」+ Web削除ページ）
- 削除リクエストURL: https://tatsunoritojo.github.io/tsuzumi/data-deletion.html
