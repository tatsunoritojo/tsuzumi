データ削除のご案内

最終更新日：2026年2月19日

Tsuzumi（以下「本アプリ」）では、ユーザーがいつでも自分のデータを完全に削除できる機能を提供しています。

アプリ内からデータを削除する方法

  1. Tsuzumi アプリを開きます
  2. 画面下部の「設定」タブをタップします
  3. 「アカウント削除」をタップします
  4. 確認画面で「削除する」をタップします

削除されるデータ

アカウント削除を実行すると、以下のすべてのデータが完全に削除されます。

  • ユーザーアカウント情報（匿名ユーザーID）
  • すべての習慣カードと記録ログ
  • エール（応援メッセージ）の送受信履歴
  • お気に入りデータ
  • ニックネーム・通知設定・その他の設定データ
  • FCMトークン（プッシュ通知用デバイストークン）
ご注意：アカウント削除後、データの復元はできません。

保持されるデータ

アカウント削除後に保持されるデータはありません。すべてのデータはサーバーから即時に削除されます。
