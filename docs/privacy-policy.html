プライバシーポリシー

最終更新日：2026年2月15日

Tsuzumi（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーでは、本アプリが収集する情報、その利用方法、およびユーザーの権利について説明します。

1. 収集する情報

本アプリは以下の情報を収集します。

1-1. 自動的に収集される情報

  • 匿名ユーザーID：Firebase Authentication により自動生成される匿名の識別子です。メールアドレス、電話番号、氏名などの個人を特定できる情報は収集しません。
  • FCMトークン：プッシュ通知の配信に使用するデバイストークンです。
  • タイムゾーン情報：通知の適切な時刻配信のために使用します。

1-2. ユーザーが入力する情報

  • 習慣データ：ユーザーが作成した習慣カードの名称、カテゴリ、記録ログなど。
  • ニックネーム：ユーザーが任意で設定する表示名（20文字以内）。
  • 通知設定：通知モード、配信時刻、お休みモードなどの設定値。

2. 情報の利用目的

収集した情報は以下の目的で利用します。

  • 習慣トラッキング機能の提供
  • エール（応援メッセージ）の送受信
  • プッシュ通知の配信
  • アプリの改善および不具合の修正

3. 情報の共有

本アプリでは、ユーザーが習慣カードを「公開」設定にした場合に限り、以下の情報が他のユーザーに表示される場合があります。

  • 習慣カードの名称
  • 継続日数
  • ニックネーム（設定している場合）

上記以外の目的で、ユーザーの情報を第三者に販売・提供することはありません。

4. 第三者サービスの利用

本アプリは、Google LLC が提供する Firebase を利用しています。

  • Firebase Authentication：匿名認証によるユーザー管理
  • Cloud Firestore：データの保存・同期
  • Firebase Cloud Messaging (FCM)：プッシュ通知の配信
  • Firebase Crashlytics：アプリのクラッシュレポート収集（個人を特定できない技術情報のみ）

Firebase のプライバシーポリシーについては、Firebase プライバシーに関する情報をご確認ください。

5. データの保管

ユーザーのデータは Google Cloud のサーバーに保管されます。データの保管期間は、ユーザーがアカウントを削除するまでです。

また、アプリの動作に必要な以下の情報がデバイス上にローカル保存されます。

  • 認証セッション情報（自動ログイン維持のため）
  • 初回起動フラグ（オンボーディング表示制御のため）

6. データの削除

ユーザーは、アプリ内の「設定」→「アカウント削除」から、いつでもアカウントおよびすべての関連データを完全に削除できます。削除されるデータには以下が含まれます。

  • ユーザーアカウント情報
  • すべての習慣カードと記録ログ
  • エールの送受信履歴
  • お気に入りデータ
  • 通知設定およびその他の設定データ

アカウント削除後、データの復元はできません。

7. 子どものプライバシー

本アプリは全年齢のユーザーを対象としています。13歳未満の児童が利用する場合は、保護者の同意のもとでご利用ください。本アプリは個人を特定できる情報を収集しないため、児童のプライバシーリスクは最小限です。

8. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新される場合があります。重要な変更がある場合は、アプリ内で通知します。
